# 📚 StudyMate — Smart Student Planner

A desktop application built with Java and JavaFX that helps students
manage their academic life.

## Features
- 🏠 Dashboard with live stats and upcoming deadlines
- 📝 Assignment tracker with priority levels and countdown timers
- 📅 Weekly timetable view
- 💾 Automatic data saving
- ⚠️ Overdue assignment alerts

## Technologies Used
- Java 17
- JavaFX 17
- File I/O for data persistence

## How to Run
1. Clone this repository
2. Open in IntelliJ IDEA
3. Add JavaFX 17 SDK to your project libraries
4. Run Main.java
